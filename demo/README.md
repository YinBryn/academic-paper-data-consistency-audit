# Demo

This demo uses synthetic data only. It is not based on any real paper.

The example shows how a small table of temperature-dependent resistance values can be checked using the Arrhenius fitting script. The output is written as a short, neutral technical note rather than an accusation.

Run:

```bash
python ../scripts/arrhenius_fit.py --temperature-c 800 750 700 --resistance 0.025 0.052 0.110
```
