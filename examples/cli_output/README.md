# CLI Output Examples

This directory records representative output from the `paper-audit` CLI using synthetic inputs.

The goal is to help users understand what the tool prints before installing or running it locally.

All examples are synthetic and are not based on any real paper, author, DOI, journal, or publisher-provided data set.

## Examples

| File | Command |
|---|---|
| `arrhenius_output.txt` | `paper-audit arrhenius` |
| `statistics_output.txt` | `paper-audit statistics` |
| `ratio_output.txt` | `paper-audit ratio` |
| `dimensional_output.txt` | `paper-audit dimensional` |
