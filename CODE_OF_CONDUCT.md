# Code of Conduct

## Purpose

This project supports technical, reproducible, and neutral discussion of data consistency in scientific papers. Contributions should help improve clarification, reproducibility, and evidence-claim alignment.

## Expected behavior

Participants are expected to:

- use professional and respectful language
- focus on data, calculations, evidence, and documentation
- distinguish observation, recalculation, and interpretation
- avoid inferring author intent
- avoid accusatory or inflammatory language
- respect copyright, privacy, and data-use restrictions
- welcome good-faith technical correction

## Unacceptable behavior

The following are not acceptable in this project:

- harassment or personal attacks
- allegations of misconduct based only on repository discussions
- publication of private personal information
- uploading copyrighted PDFs or proprietary datasets without permission
- repeated off-topic or inflammatory comments
- deliberate misrepresentation of calculations or evidence

## Scope

This code of conduct applies to issues, pull requests, discussions, documentation, examples, and any project-related communication.

## Enforcement

Maintainers may edit, close, or remove content that violates these expectations. Repeated or serious violations may result in restricted participation.

## Reporting

Report concerns through GitHub issues or, for sensitive matters, through GitHub's private security/advisory channels if available.
