# GitHub Pages Setup

The documentation is designed to work with GitHub Pages using the `/docs` folder.

## Suggested settings

In the GitHub repository:

1. Open **Settings**.
2. Open **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch: `main`.
5. Select folder: `/docs`.
6. Save.

After GitHub Pages finishes building, the documentation site should use `docs/index.md` as the homepage.

## Notes

- The repository README remains the main GitHub landing page.
- The GitHub Pages site is intended as a cleaner documentation and sharing surface.
- The documentation uses only repository Markdown files and synthetic examples.
