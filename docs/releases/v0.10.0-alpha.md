# v0.10.0-alpha Release Notes

## Summary

`v0.10.0-alpha` is the first consolidated alpha release of **Academic Paper Data Consistency Audit** as an installable command-line toolkit.

The project provides physics-informed, reproducible checks for technical consistency in materials electrochemistry papers. It is designed for neutral clarification, source-data comparison, and evidence-claim alignment. It does not infer author intent and does not classify misconduct.

## Highlights

- Installable `paper-audit` CLI
- Arrhenius fitting and activation-energy recalculation
- Mean and standard-deviation recalculation
- Performance ratio calculation
- I–V–P consistency checks
- Batch reported-value/source-data tolerance reports
- Rp/ASR component-sum consistency checks
- Faradaic-efficiency and gas-production consistency checks
- Conductivity geometry-normalization checks
- Synthetic case studies for audit workflows
- Neutral PubPeer-style comment templates
- Responsible-use and contribution guidance
- GitHub Pages documentation foundation

## CLI Commands

```bash
paper-audit arrhenius
paper-audit statistics
paper-audit ratio
paper-audit dimensional
paper-audit tolerance-report
paper-audit resistance-sum
paper-audit faradaic-efficiency
paper-audit conductivity-geometry
```

## Synthetic Case Studies

The release includes synthetic examples for:

- activation-energy recalculation
- I–V–P consistency
- batch table/source-data tolerance reporting
- Rp/ASR component-sum consistency
- Faradaic-efficiency consistency
- conductivity geometry normalization
- figure/table/source-data mismatch
- evidence-claim alignment

All examples are synthetic and are not based on any real paper, author, DOI, journal, or publisher-provided data set.

## Responsible-use note

This project supports technical clarification and reproducibility-oriented review. It should be used to formulate precise, evidence-based questions. It should not be used to infer author intent or make misconduct allegations.
