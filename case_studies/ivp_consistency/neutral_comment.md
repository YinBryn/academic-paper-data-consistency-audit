# Neutral Comment Draft

I would appreciate clarification regarding the power-density value listed for `Synthetic-Cell-B`.

The table reports a current density of 2.00 A/cm² and a voltage of 0.70 V. Using the direct relation `P = j × V`, these values correspond to 1.40 W/cm². This appears to differ from the reported power density of 1.10 W/cm².

Could the authors clarify whether the current density, voltage, and power-density values refer to the same operating point, or whether an additional correction or normalization was applied?
