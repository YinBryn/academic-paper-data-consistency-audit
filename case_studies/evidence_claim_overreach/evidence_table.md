# Synthetic Evidence Table

This case demonstrates evidence-claim alignment using a fully synthetic mechanistic example.

## Synthetic claim

> The improved fuel-cell performance is caused by faster proton transport through the bulk lattice.

## Available synthetic evidence

| Evidence item | Synthetic observation | Directly supports the claim? |
|---|---|---|
| I–V curve | Higher peak power density for Sample B than Sample A | Partially; shows performance difference, not transport mechanism |
| EIS | Lower total polarization resistance for Sample B | Partially; suggests improved electrode/cell kinetics, not necessarily bulk proton transport |
| XRD | Same major perovskite phase before and after testing | No direct support for faster proton transport |
| SEM | Similar porous electrode morphology | No direct support for bulk proton transport |
| Conductivity isotope test | Not provided | Missing |
| Proton transport number | Not provided | Missing |
| Hydration thermodynamics | Not provided | Missing |

## Technical interpretation

The available evidence supports improved electrochemical performance, but it does not by itself isolate faster bulk proton transport as the causal mechanism.
